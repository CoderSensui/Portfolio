LibNav

Library Navigation System

A web-based map navigation system for Notre Dame-Siena College of GenSan. Students find books, admins broadcast announcements — all powered by Firebase with zero backend server.

Interactive floor maps, real-time book search via Firebase RTDB, admin panel with maintenance mode toggle, broadcast system, Firebase Auth, and service workers for offline support. Built solo as a capstone project.

Vanilla JS Firebase RTDB Firebase Auth Vercel Service Worker
